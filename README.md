# travelapp

欢迎fork, 转载请联系作者 izhouwl@163.com
用Vue开发旅游app

技术栈：Vue.js \ vuex \ vue-router \ axios \ better-scroll
其他： H5 localStorage \

## 项目亮点

* 首页
1. Header栏
2. Iconfont在线字体
3. 首页轮播图
4. 图表栏
5. 热销推荐、周末游组件

* 城市选择页
1. 城市搜索框
2. 右侧字母表布局
3. 列表

* 详情列表页与详情页
1. banner布局
2. 共用画廊组件
3. header栏，渐显渐隐效果
4. 画廊动画效果
